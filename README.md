# Text Encryption and Decryption System

## Overview

This project implements a simple text encryption and decryption system using the Caesar cipher algorithm in Python. The program allows users to encrypt and decrypt messages by shifting each alphabetic character by a specified number of positions.

## Features

* Encrypt text using a custom shift key.
* Decrypt encrypted text.
* Supports both uppercase and lowercase letters.
* Preserves spaces, numbers, and special characters.
* User-friendly command-line interface.
* Allows multiple encryption and decryption operations in a single run.

## How It Works

The Caesar cipher is a substitution cipher in which each letter in the message is shifted by a fixed number of positions in the alphabet.

Example:

* Original text: `Hello World`
* Shift key: `3`
* Encrypted text: `Khoor Zruog`

## Technologies Used

* Python

## How to Run

1. Clone the repository or download the source code.
2. Open the project folder in VS Code.
3. Run the following command:

```bash
python project2.py
```

4. Enter the text and shift key when prompted.
